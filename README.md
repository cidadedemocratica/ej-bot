# EJ Bot




# Primeiros passos

```
make first-run
```


# Fluxo de trabalho

Para realizar o treinamento e criar os modelos necessários para conversa do bot é realizado o comando:

```
make train
```

Para utilização do bot em terminal há o comando

```
make run-shell
```
